# Market Brief: Traditional Chinese (zh-tw) — Taiwan

## Buyer persona

Taiwanese bag/backpack brand owners, trading companies, and sourcing managers.
Taiwan has a mature bag OEM heritage (many TW-owned factories moved to
Vietnam/China decades ago), so readers are industry-literate and evaluate
factories like insiders: they ask about 打樣 quality, 交期 reliability,
產能, and QC process before price.

## What to emphasize

1. Japanese-brand track record (Anello, 15+ years) — Taiwan buyers deeply
   respect Japanese QC standards; this is the lead trust signal.
2. Myanmar cost advantage vs China/Vietnam — TW buyers actively diversify
   out of China; frame Myanmar as the next rational step with concrete
   numbers, not slogans.
3. Process transparency: 2 週打樣, 24 小時回覆, AQL 檢驗, 認證(ISO 9001,
   OEKO-TEX, GRS).

## Language rules (this is localization, NOT charset conversion)

Source text is Simplified Chinese (zh.json). Rewrite in Taiwan Mandarin
conventions — vocabulary differs from mainland usage:

| 大陆用语 | 台灣用語 |
|---|---|
| 质量 | 品質 |
| 信息 | 資訊 |
| 背包(时尚语境) | 後背包(通称仍可用 背包/包款) |
| 定制/定製 | 客製化 |
| 出货 | 出貨 |
| 打样 | 打樣 |
| 拉链 | 拉鍊 |
| 文件(file upload) | 檔案 |
| 视频 | 影片 |
| 网络 | 網路 |
| 项目(project) | 專案 |
| 软件/硬件 | 軟體/硬體 |
| 通过(认证) | 通過/取得(認證) |
| 支持 | 支援(技术)/支持(抽象) |
| 用户 | 使用者/客戶 |
| 反馈 | 回饋 |
| 联系 | 聯絡/聯繫 |
| 周一至周五 | 週一至週五 |
| 小贴士 | 小提醒 |

- Punctuation: full-width Taiwan conventions;quotes use 「」 not “”。
- Keep OEM, ODM, MOQ, QC, Tech Pack, AQL, FOB/CIF/EXW, YKK, Pantone,
  ISO 9001, OEKO-TEX, GRS in Latin script (Taiwan B2B convention).
- 代工 is the natural Taiwanese word for OEM manufacturing: "OEM 代工" reads
  native; use it where the simplified source says "OEM 生产/代工生产".
- Brand names, addresses, emails, phone numbers unchanged.
- Numbers/facts (2003, 20+, 600+, 24 小時, 2 週, 5 檔案, 10MB) unchanged in value.

## Tone

Professional and warm-direct, the way Taiwanese B2B copy reads: courteous
(請/歡迎/竭誠) without mainland-style slogan cadence. Avoid 网感 phrasing.
